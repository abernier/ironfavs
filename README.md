Customise your `*.ironhack.com`'s favicons with an emoji.

# INSTALL

Pre-requisites:
  - install the [Tampermonkey](https://www.tampermonkey.net) extension, for example for [Chrome](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo).

Go to [ironfavs.user.js](https://github.com/abernier/ironfavs/raw/master/ironfavs.user.js) and click the `Install` button:
![](https://assets.codepen.io/67030/image_13.png)
